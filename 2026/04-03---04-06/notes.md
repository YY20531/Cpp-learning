## ew
- 多字符常量
       - 'ab' 不是字符，也不是字符串。
  - 它是 多字符常量（multi-character literal）。
    - 类型是 int，不是 char。
      - 编译器会把每个字符的 ASCII 值拼成一个整数（具体方式依赖于系统的字节顺序，通常是 大端或小端）。
  
        | 类型     | 单引号/双引号 | 可包含     | 类型                   | 输出行为      |
        | ------ | ------- | ------- | -------------------- | --------- |
        | char   | `' '`   | 1 个字符   | char                 | 输出该字符     |
        | int    | `'ab'`  | 2~4 个字符 | int                  | 拼成整数，输出整数 |
        | string | `"ab"`  | 任意字符    | const char* / string | 输出字符序列    |
