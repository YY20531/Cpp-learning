# STL 初步了解
>Standard Template Library

>别说，又和模板串起来了，得劲

<details>
<summary>C++ 标准库..</summary>

│

├── STL（标准模板库）⭐⭐⭐

│   ├── 容器（vector / set / map / ...）

│   ├── 算法（sort / find / ...）

│   ├── 迭代器（begin / end）

│

├── 输入输出（iostream）

├── 字符串（string）

├── 数学（cmath）

├── ...
</details>

<details>
  <summary> 👉 STL = 容器（存数据） + 算法（处理数据） + 迭代器（连接两者）</summary>

- 🧱 容器：负责存数据
- 🔗 [迭代器](./lterator.md/)：统一访问方式
- ⚙️ 算法：操作“区间”，而不是容器本身
</details>
