# STL 初步了解
>Standard Template Library

>别说，又和模板串起来了，得劲

C++ 标准库

│

├── STL（标准模板库）⭐⭐⭐

│   ├── 容器（vector / set / map / ...）

│   ├── 算法（sort / find / ...）

│   ├── 迭代器（begin / end）

│

├── 输入输出（iostream）

├── 字符串（string）

├── 数学（cmath）

├── ...


- 👉 STL = 容器（存数据） + 算法（处理数据） + 迭代器（连接两者）
- 模板实现，***泛型编程***

### 包括但不限于
- [vector<T>](/2026/03-28/notes.md/)
- sort
- map<K, V>
- 。。。。。
